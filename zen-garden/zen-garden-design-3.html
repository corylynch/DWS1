Design #3

Aside from doing basic changes like altering the text, background color etc., this website placed an image with a cut-out in the middle. This is where the content is held on the page. They positioned the main image appropriately with css, then they set the position to fixed. ex. "Position: fixed". So when scrolling down, the image in front does not move, but only the content does.
