Design #1

I like this website design as it's very unique. Upon entrance, a robot illustration emerges from behind a semi transparent background which contains the page's main content, all in front of a spinning illustration in the very background. I believe that I know how this 'movement' on the page is achieved, but let's first observe the basics:

  • - The large header was given a new font in CSS, enlarged, then restyled with a new color. If any other <h1>'s were used on the web page, then the main header at the top was specifically targeted using an ID.
  • - All paragraphs seem to be styled the same on this page. This means that the CSS documents targets the paragraphs by simply using a <p> tag.
  • - All of the links on the page are also styled the same way with an exception of a few. The few that were intended to be designed differently than the other must have been put in a class to 'override' the previous design that was given to them.

The main moving elements of the page are illustrations that used a <img> tag to be brought into the site, however they are styled with CSS. First, they we're put in the far back by being assigned a z-index to them, one in front of the other. I think the movement upon loading the web page is done by using the 'transition: ' property.
